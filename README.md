# Alzaf Store Frontend

A Next.js e-commerce frontend built for the Alzaf IT developer task.

## Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS + Custom CSS Variables
- **Language:** TypeScript

## Getting Started

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run development server:**
   ```bash
   npm run dev
   ```

3. **Build for production:**
   ```bash
   npm run build
   npm start
   ```

## Features
- **Server-Side Rendering (SSR):** Parallel fetching for banners and products.
- **Responsive Design:** Mobile drawer navigation,, carousel, and optimized grid.
- **Advanced Filtering:** URL-based filtering for categories, price, search, and sorting.
- **Error Handling:** hndle Custom 404 and error boundaries (includes specific handling for Product ID 1).
